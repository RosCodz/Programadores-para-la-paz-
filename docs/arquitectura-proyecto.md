# Arquitectura del Proyecto

## Nombre

Plataforma de Orientación y Peticiones Ciudadanas

## Componentes

### Node.js
Controla la aplicación web, formularios, login simple, rutas y comunicación con n8n.

### n8n
Automatiza el flujo de generación del derecho de petición.

### DeepSeek
Mejora la redacción del borrador, sin inventar hechos ni reemplazar revisión jurídica.

### Google Docs
Entrega un documento editable.

### Google Sheets
Registra el seguimiento de solicitudes.

### Telegram
Notifica al equipo sin datos sensibles.

### ClickUp
Gestiona el backlog Scrumban, tareas y evidencias.

### GitHub
Guarda código, ramas, commits y documentación técnica.
