# Backlog Scrumban - Plataforma de Orientación y Peticiones Ciudadanas

## Objetivo

Construir una plataforma de apoyo pedagógico para generar borradores de derechos de petición,
orientar rutas ciudadanas y registrar seguimiento mediante herramientas digitales.
