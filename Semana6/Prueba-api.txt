Al enviar una petición POST a http://localhost:3000/registro con el
siguiente JSON:
{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor responde con:
{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto confirma que el servidor recibe correctamente los datos enviados
por el usuario y los devuelve organizados en formato JSON.
